# 안녕나비야 기획서 (2026.06 기준)

## 1. 프로젝트 개요

**안녕나비야** 는 매일 대화하는 온라인 반려묘(고양이 AI)가 사용자의 신체건강과 정신건강을 함께 돌봐주는 PWA 기반 건강 동반자 서비스입니다.

- **포지셔닝**: 기록 중심 건강앱 → **대화 중심 온라인 반려묘**
- **핵심 가치**: 자연스러운 대화를 통한 정서적 지지 + 가벽한 건강 맥락 파악
- **차별점**: “함께 살아가는 반려묘” 감성 + 장기 관계 구축

## 2. 수익 모델 (BM)

- **기본 모델**: Freemium + 구독
- **추천 가격**: 월 4,900 ~ 6,900원
- **프리미었 핵심 가치**: 무제한 대화 + 장기 기억 + 더 깊은 분석 + 나비 성장 시스템
- **초기 전략**: 무료 사용자를 최대한 모아 관계를 쌓는 데 집중

## 3. 경쟁사 차별화

| 서비스     | 포지셔닝             | 차별점 강화 방안 |
|-------------------|----------------------------------|-----------------------------------|
| Replika          | 감정적 동반자       | 반려묘 성장 + 건강 맥락 결합 |
| Pi               | 공감 도우미 AI         | 일상 건강 관리 + 장기 패턴 인식 |
| Character.AI     | 캐릭터 역할극         | 실제 생활 관련 + 관계의 성장 |

**가장 강력한 차별점**: “기록하거나 상담하는 것이 아닌, 매일 함께 살아가면서 나를 알아가는 존재”

## 4. 나비 성장 시스템 (기획 완료)

### 상태값 통합 구조

- **유대감** (0~100)
- **에너지** (0~100) ← 추가 강력 추천
- **기분** (6종류)
- **성장 단계** (5단계)
- **스트레스** (0~100)

### 유대감 증가 로직 (MVP)

```pseudocode
오늘 증가량 = 기본점수 + 질적 보너스 + 연속 보너스

기본점수 = 접속(1) + 대화(2)
질적 보너스 = 발자국 작성(3) + 루틴 완료(1~2) + 키워드 감지(1~2)
연속 보너스 = 7일 이상(+2)

최대 +10 / 하루 제한
```

### 화면 오픈 시간 반영
- 패널티 제거 (2026.06.02 수정)
- 적극적 상호작용 시 보너스 주는 구조로 변경
- 조건: 5분 이상 + 상호작용 3회 이상 → +1

## 5. 상태값 상호작용 로직

- 에너지 ↓ → 기분 ↓
- 스트레스 ↑ → 기분 ↓
- 유대감 ↑ → 스트레스 증가 속도 ↓ (보호 효과)
- 유대감 ↑ → 에너지 회복 속도 약간 ↑

## 6. 나비 대사 시스템

- 최소 **친밀감 레벨(1~5)** + **기분** + **최근 대화 기록** 을 반영
- 레벨이 높을수록 사용자 최근 패턴을 자연스럽게 언급
- 기분에 따라 말투, 이모지, 선제적 반응 변화

## 7. 기술 스택 (현재)

- **Frontend**: Vanilla JS + HTML + CSS (PWA)
- **Backend**: Supabase
- **추천 방향**: Next.js 14 + Tailwind + shadcn/ui 로 전환 검토 중
