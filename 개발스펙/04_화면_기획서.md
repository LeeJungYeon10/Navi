# 안녕나비야 - 화면 기획서 (Wireframe Spec)

## 디자인 원칙
1. **Minimal & Spatial**: 복잡한 UI 배제, 공간의 여백으로 심리적 안정감 제공
2. **Conversational Focus**: 대화에 온전히 몰입할 수 있는 메신저 형태
3. **Subtle Interaction**: 화려한 이펙트보다는 배경 톤의 미세한 변화, 은은한 빛 효과로 보상 표현

---

## 1. 스플래시 & 온보딩 (Guest Mode)

**목적**: 로그인 장벽 없이 즉시 나비와 대화 시작

**UI 구성**:
- 배경: 새벽녘의 고요한 분위기 (다크 모드 기본)
- 중앙: 미세하게 박동하는(Breathing) 추상적인 나비 그래픽
- 텍스트: 1.5초 후 타이핑 애니메이션으로 "오늘 나는 널 만나서 기분이 좋아. 너의 기분은 어때?"
- 하단 입력창 + 전송 아이콘
- 입력창 위: 가벼운 툴팁 ("여기에 오늘의 기분을 편하게 적어주세요.")

---

## 2. 메인 대화 화면 (Home - Chat)

**목적**: 서비스의 핵심 공간. 대화 + 데이터 수집 + 행동 유도

**UI 구성**:
- 상단 바: 중앙에 작은 '안녕나비야' 타이포, 우측 햄버거 메뉴 (마이페이지/정원 이동)
- 대화 스트림:
  - 사용자 말풍선: 우측 정렬
  - 나비 말풍선: 좌측 정렬 (테두리 없이 텍스트만 또는 가벼운 Glassmorphism)
- 퀵 리플라이 칩: 입력창 바로 위, 가로 스크롤 가능한 알약 버튼
  - 예: [응, 창문 열고 올게!], [조금 이따가 할래]
- 시각적 피드백: 건강 행동 완료 시 배경 특정 영역(나비 그래픽 또는 달의 위상)이 미세하게 밝아지거나 색조가 따뜻하게 변하는 트랜지션 (1~2초)

---

## 3. 로그인 유도 바텀 시트

**목적**: 대화의 맥락을 끊지 않으면서 자연스럽게 회원가입 유도

**UI 구성**:
- 오버레이: 대화 화면 40% 딤 처리
- 바텀 시트 (하단에서 부드럽게 올라옴)
- 타이틀: "우리의 대화를 나비 정원에 보관할까요?"
- 서브 텍스트: "로그인하면 언제든 이전 대화와 감정의 변화를 다시 볼 수 있어요."
- 버튼:
  - Primary: [Google로 계속하기] (구글 로고 포함)
  - Secondary: [조금 더 대화할래] (텍스트 버튼)

---

## 4. 나비 정원 (History & Garden) - 로그인 전용

**목적**: 과거 기록을 감성적으로 확인 (단순 리스트가 아닌 갤러리 느낌)

**UI 구성**:
- 상단 요약: "이번 주 수면 시간이 평균 6.8시간으로 늘었어요. 꾸준히 관리하면 컨디션이 더 안정적일 거예요." (사실 기반 + 부드러운 격려)
- 타임라인 리스트 (세로 스크롤):
  - 날짜별 카드
  - 카드 내부: 날짜, 대표 감정 태그 (#차분함, #가벼운피로), 수행한 행동 아이콘 (물방울, 걷기 등)
  - 아이디어: 카드 배경색이 그날의 주요 감정에 따라 파스텔 톤으로 미세하게 다르게 렌더링

---

## 5. 마이페이지 (Settings)

**목적**: 최소한의 설정만 직관적으로 제어

**UI 구성**:
- 프로필 섹션: 연동된 Google 이메일, 닉네임 수정 필드 (나비가 부르는 이름)
- 설정 리스트:
  - 푸시 알림 ON/OFF (토글)
  - 테마 설정 (라이트 / 다크 모드)
- 하단 액션:
  - 로그아웃 (텍스트 버튼)
  - 서비스 탈퇴하기 (연한 회색, 눈에 덜 띄게)
